# Chapter 0：Lab Introduction

# 实验室介绍与学习指南

---

# 1. 欢迎加入实验室

欢迎加入 **Multimodal Document AI Laboratory**。

本实验室主要研究多模态大模型（Vision Language Model，VLM）及其在文档智能（Document AI）领域的理论、算法与应用，重点关注文档理解、OCR、版面分析、表格识别、公式识别、结构化解析以及智能 Agent 等方向。

我们的目标不仅是完成科研论文，更希望培养具有独立科研能力和工程实践能力的 AI 研究人员。

---

# 2. 实验室研究方向

目前实验室主要开展以下几个研究方向：

## （1）Vision Language Model（VLM）

典型模型包括：

* Qwen3.5-VL
* InternVL
* SmolVLM
* Florence
* Molmo

学习目标：

理解现代视觉语言模型的基本原理及应用。

---

## （2）Document AI

主要研究内容：

* OCR
* Layout Analysis
* Reading Order
* Table Recognition
* Formula Recognition
* Chart Understanding
* Markdown Generation
* Document Parsing

学习目标：

理解现代文档智能系统整体架构。

---

## （3）大模型微调（Fine-tuning）

主要内容：

* LoRA
* QLoRA
* PEFT
* Instruction Tuning
* SFT

学习目标：

能够独立完成模型微调。

---

## （4）Benchmark 与模型评测

主要内容：

* OmniDocBench
* DocLayNet
* OCRBench
* DocVQA

学习目标：

能够科学评价模型性能。

---

# 4. 实验室学习方式

实验室采用 **"学习 + 实践 + 科研"** 的培养模式。

学习流程如下：

```text
官方文档
        │
        ▼
阅读源码
        │
        ▼
运行 Demo
        │
        ▼
修改代码
        │
        ▼
开展实验
        │
        ▼
科研创新
```

每一步都必须亲自动手完成。
